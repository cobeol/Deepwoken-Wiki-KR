# Races

종족은 기본 스텟과 <Link to="/talents#innate">고유 재능</Link>을 얻는 딥워큰의 요소입니다.<br/>
슬롯 생성 시, 종족이 정해지며 이를 바꾸려먼 150로벅스를 내야합니다. <ColorText color="#CD0C22"><b>(추천하지 않습니다. 슬롯 구매는 200로벅스, 종족 리롤은 150로벅스)</b></ColorText>

-   기본 스텟은 두 종류의 스텟을 각각 2씩 올려줍니다.
-   하지만, 이는 대부분의 빌드에서 의미를 가지지 않습니다. (물론 일부 빌드는 종족이 중요할 수도 있습니다)
