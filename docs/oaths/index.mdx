# Oaths

## 설명

"맹세"는 캐릭터가 쓰러지는 날까지 살아가겠다고 맹세하는 관점과 신념입니다. <br/>
플레이어가 특정 요구 사항을 충족해야만 얻을 수 있는 고유한 재능 카드로 분류됩니다.

맹세는 독특한 특성으로 하나만 맺을 수 있습니다. <br/>
